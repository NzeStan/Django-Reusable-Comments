# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-04-26

### Added
- Initial release of django-reusable-comments
- Core Comment and CommentFlag models
- REST API for creating, retrieving, updating, and deleting comments
- Moderation workflow with approval/rejection capabilities
- Flagging system for inappropriate comments
- Customizable settings
- Support for both UUID and integer primary keys
- Threaded comments with hierarchical structure
- Django admin integration with custom admin actions
- Signal system for extending functionality
- Comprehensive test suite
- Documentation
